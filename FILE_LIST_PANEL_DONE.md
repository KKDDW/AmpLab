## 📝 代码特点

### 职责清晰
```python
# UI 层：只负责显示和事件转发
class FileListPanel:
    def refresh(self, file_list):  # 接收数据
        ...
    def _on_add_clicked(self):      # 转发事件
        self.on_add_files()

# 业务层：处理逻辑
class AppDispatcher:
    def _on_add_files(self):        # 业务逻辑
        ...
        self.ui.refresh_file_list(self.file_list)  # 通知 UI
```

### 易于扩展

# 未来添加功能只需：
